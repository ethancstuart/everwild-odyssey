# Alpha HUD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a lightweight in-engine HUD that displays hero health, resource, level, XP, and control reminders during the alpha slice.

**Architecture:** Implement `AEOHUD` using Unreal's `AHUD::DrawHUD` for a low-dependency overlay. Keep the display text generated from a testable snapshot struct so the UI can later migrate to UMG without losing the data contract.

**Tech Stack:** Unreal Engine 5.7, C++, `AHUD`, existing hero gameplay components, Unreal automation tests.

    Snapshot.Health = 120.0f;
    Snapshot.MaxHealth = 135.0f;
    Snapshot.Resource = 65.0f;

    Snapshot.Experience = 25;
