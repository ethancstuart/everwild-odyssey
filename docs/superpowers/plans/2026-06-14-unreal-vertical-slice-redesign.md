# Unreal Vertical Slice Redesign Plan

## Phase 1: Lock the bigger slice contract

- Expand alpha world scaffold tests for landmark, scenic prop, and ambient light counts.
- Expand game mode tests for a larger opening enemy formation.
- Expand HUD tests for objective and world-event tracker formatting.

## Phase 2: Build the valley presentation

- Replace the six-piece scaffold with a runtime-authored alpha valley assembled from Unreal built-in meshes and dynamic materials.
- Add terrain plates, roadways, lake water, bridge spans, village pieces, keep pieces, ruins, forest clusters, crystal surges, and ambient lights.
- Add sun, skylight, and fog components on the scaffold actor for a stronger default frame.

## Phase 3: Make the opening loop playable

- Spawn a larger relic surge formation with some stronger silhouettes.
- Give enemies simple chase/contact behavior so the scene is alive.
- Add visible companion silhouettes to the hero.
- Let defeated enemies award XP, relic shards, and eventual gate stabilization.

## Phase 4: Upgrade the player read

- Add quest/objective/world-event lines to the HUD.
- Keep keyboard and controller controls visible in a compact tracker.
- Compile, launch, visually inspect, and iterate if the first frame still undersells the slice.
